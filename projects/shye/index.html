ANONYMOUS NETWORK SECURITY PROTOCOL

Shye

ACTIVE RESEARCH

Shye 是一个去中心化匿名网络安全协议研究计划。

其核心目标并非维持长期稳定通信， 而是通过短生命周期路径、 动态竞拍、 一次性公钥、 局部信任机制与持续拓扑扰动， 降低网络长期中心化与路径固化风险。

Shye 不试图构建“永久匿名线路”。 相反，它倾向于构建： 一个持续变化、 持续重组、 持续失稳的匿名通信系统。

CORE PRINCIPLES

Short-lived Communication

不鼓励长时间稳定通信。 长期路径会逐渐形成中心化节点与稳定观察面。

Dynamic Auction Routing

节点通过动态竞拍竞争发包权， 避免固定转发结构长期存在。

Topology Disturbance

每轮随机引入部分新节点参与竞拍， 持续扰动网络结构。

Local Trust

节点仅维护局部行为信任。 丢包会降低权重， 但不会直接永久拉黑。
